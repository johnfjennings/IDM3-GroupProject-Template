# IDM3 Sample Intellij Project

## Introduction
This repo contains a sample Intellij project with a Docker container deployment configuration. 

The root folder of this repo cobntaibns the following:
* docs - Please place any Word, Excel, Powerpoint, etc. files here
* StaticWebsite - Please use this folder when designing your site template and us it to contain a static html version of your site
* Intellij - This is the main Intellij project folder and should be opened with Intellij Ultimate Edition 2024.x+.
    - This Intellij root folder also contains the following non-Intellij related folder:
        * Docker - This folder contains all the docker related files for deployment 
 

## Requirements

Install the following software:
* Intellij Ultimate Edition 2024 (https://www.jetbrains.com/idea/)  
* Docker-desktop (https://docs.docker.com/docker-for-windows/)
* Git (https://git-scm.com/)
